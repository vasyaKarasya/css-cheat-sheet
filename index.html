Basic CSS selectors

CSS selectors define the pattern to select elements to which a set of CSS rules are then applied

CSS selectors can be grouped into the following categories based on the type of elements they can select.

BASIC CSS SELECTORS
Type of selector Description Syntax Example
Universal selector Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces. * ns|* *|* * will match all the elements of the document.
Type selector Selects all elements that have the given node name. elementname input will match any <input> element.
Class selector Selects all elements that have the given class attribute. .classname .index will match any element that has class="index"
ID selector Selects an element based on the value of its id attribute. There should be only one element with a given ID in a document. #idname #start-button will match the element that has id="start-button".
ID selector Selects all elements that have the given attribute. [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value] [autoplay] will match all elements that have the autoplay attribute set (to any value).
